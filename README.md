Para poder poner el texto en negrita, este deber  ser encerrado entre doble **.

###5. Cuadros para códigos o reseñas (BLOCKQUOTES)

Estos documentos son utilizados para resaltar instrucciones específicas para la instalación, configuración y/o inicializacion o mostrar secciones de código fuente. Se maqueta iniciando el texto con un símbolo de mayor que (>)

**Ejemplo:**

Para listar las carpetas y archivos desde una terminal de sistemas operativos Windows debemos ingresar el comando:

> C:/dir
Después oprimidos la tecla "Enter".
También podemos ingresar textos multilinea

**Ejemplo:**
>Aquí ingresan un conjunto de instrucciones
>para explicar al usuario, como instalar el
>Software que hemos diseñado
Y deseamos incluir viñetas para enlistar pasos podemos usar el carácter
-detro del texto a documentar

**Ejemplo:**

**Pasos para instalar la base de datos**

>-Descargar MySQL del sitio oficial
>-Instalar el sistema gestor de base de datos, definiendo el puerto y contraseña para el usuario ***root***
>-Descargamos el archivo de respaldo de la base de datos (.sql)
>- Restauramos la base de datos usando el comando *mysql*
>> C:/ProgramFiles/MYSQLServer8.0/bin/mysql-u root -p password \<respaldo.sql
